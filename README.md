# 커플력 테스트 React 버전

## 이미지 넣는 위치

아래 파일들을 `public/quizimg` 폴더 안에 넣어주세요.

```txt
public/quizimg/quiz_intro.png
public/quizimg/mainbg.png
public/quizimg/pinkbg.png
public/quizimg/main_back.png
public/quizimg/quiz_back.png
```

## 실행 방법

```bash
npm install
npm run dev
```

## 화면 구성

- Intro: 인트로
- Quiz: 문제 풀이
- Setting: 문제 만들기
- Result: 결과 페이지
